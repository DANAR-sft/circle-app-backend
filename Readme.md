Perintah untuk testing :
Jalankan semua test (unit + functional + integration + coverage):
npm test ✅
Jalankan unit test saja:
npm run test:unit
Jalankan functional test saja:
npm run test:functional
Jalankan integration test saja (butuh Docker berjalan):
npm run test:integration ⚠️ (pastikan Docker Desktop aktif)
